Выбрать из потока чисел заранее известной длинны по возможности наибольшее.

Входные данные:

total_gifts - число подарков в мешке – int
bag_of_santa_claus - мешок Деда Мороза – generator, returns float (ценность подарка в глазах Вани, положителное)
accept – принять подарок - callback function

Выходные данные: Нет.

Примеры реализации (они не решают задачу, а только демонстрируют API):

Первый:

def choose_good_gift(total_gifts, bag_of_santa_claus, accept_gift):
    for gift in bag_of_santa_claus():
        if gift > 9.:  # I know max(gifts) :)
            accept_gift()
            return

Второй:

def choose_good_gift(total_gifts, bag_of_santa_claus, accept_gift):
    for i, gift in enumerate(iter(bag_of_santa_claus())):
        if i == total_gifts // 2:  # on the middle
            accept_gift()
            return

Пример вызова с аргументами:

gifts = [0.65, 4.75, 0.21, 0.53, 0.63, 9.99, 0.81, 2.48, 0.3, 0.64]

def gift_generator():
    yield from gifts

def do_accept():
    pass

choose_good_gift(len(gifts), gift_generator, do_accept)

Как использовать:

Например при выборе места катапультирования лётчика из неисправного самолёта – чтобы безопасно приземлиться:)

Ограничения на входные данные:

10 <= total_gifts < 1000 0 < gift
